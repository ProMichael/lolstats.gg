    Minimum Requirements
    Before you install LoLStats.gg. Be sure to check out the minimum requirements below.
    • A Computer
    • A Server that has MySQL Capabilities - Ex: Wamp Server
    • A League of Legends API Key - Get it Here
    Once you have all of that, your all set to setup the application!


    Setup
    To install the SQL, you can simply create a new database called lolstats, or just execute the code below.
    CREATE DATABASE IF NOT EXISTS `lolstats`;
    We are going to create a table called apikeys, and it will have a row called id and keys. id is the primary key and auto increment on. Alternatively, just execute the code below.
    CREATE TABLE IF NOT EXISTS `lolstats`.`apikeys` (
    `id` int(11) NOT NULL AUTO_INCREMENT,
    `key` varchar(45) COLLATE latin1_swedish_ci NULL,
    PRIMARY KEY (`id`)
    ) AUTO_INCREMENT=1
    Once everything is set up, just insert your API Key into the keys field and everything should be ready to go!
